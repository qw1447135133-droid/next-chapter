# InFinio 首发支持矩阵与验收

## 正式支持

- 单首页、单会话、Agent 主导
- 原创剧本
- 参考改编
- 视频工作流
- 历史项目恢复
- 素材库与视频生产状态包
- 视频 API / CLI 通道切换与首页状态提示

## 次级能力

- 技能草案
- 维护报告
- 长对话静默压缩
- 旧路由兼容跳回首页

这些能力保留，但不作为首发主卖点，也不能干扰主流程。

## 不再使用

- 旧 workspace / 工作台页面
- 旧模块页主流程
- reverse browserview 工作流

## 首发验收命令

1. `npx tsc --noEmit --pretty false`
2. `npm run test:home-agent`
3. `npm run smoke:home-agent`
4. `npm run build -- --sourcemap`
5. `npm run electron:build`

## 关键验收场景

1. 首次打开首页时，若配置缺失，只出现首页内引导，不出现裸错误。
2. 设置补齐后，不刷新页面也能继续当前首页会话。
3. 剧本、改编、视频三条主线都在首页内进入 active 会话。
4. 历史剧本项目和历史视频项目都在首页恢复，不切到旧页面。
5. 视频通道切换为 API / CLI 后，首页状态条与实际 workflow 一致。
6. 导出生产状态包后，素材库与目录打开动作仍可继续使用。
7. Electron 可分发包可启动，并从首页直接进入一次真实会话。

## 说明

- Electron 是首发主签收面，浏览器运行继续兼容。
- 设置面板是首页内嵌配置中心，不再承载独立页面语义。
- 若后续继续扩功能，优先在首页会话内扩 action 和状态，不恢复旧工作台心智。
